Система представляет собой автоматизацию учета сотрудников нескольких компаний, обеспечивая удобный хранимый перечень сотрудников, включая их добавление, редактирование, удаление и фильтрация. Система предназначена для технических специалистов, обеспечивающих развертку, установку и последующее сопровождение системы.
